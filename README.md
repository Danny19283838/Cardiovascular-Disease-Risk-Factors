**Cardiovascular Disease Risk Factors Analysis (R)**

This project analyzes cardiovascular disease (CVD) risk factors using the CAIR CVD 2025 dataset (1,529 patients from Jamalpuri Medical College Hospital, Bangladesh).




**Skills Demonstrated**

R programming

Data cleaning & wrangling (tidyverse)

Data visualization (ggplot2)

Exploratory Data Analysis (EDA)

Reproducible workflows

Public health analytics





**Analysis Includes**

Distribution of systolic BP

Cholesterol levels by CVD status

Smoking vs CVD

Diabetes vs CVD

Physical activity comparisons
